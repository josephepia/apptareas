# TaskApp


En el desarrollo de esta app se utilizaron las siguientes herramientas

  - Ionic(Angular)
  - Material Angular
  - FlexboxGridSass
  - Firebase realTime

Tanto Ionic y Material Angular manejan sus propios estilos para el flexboxgrid, en vista de eso recurri a una libreria que me agrada, la cual es FlexboxGrid, para el dise√±o responsivo.
